## <p aligin="cenetr">Properties</p>

This sample is about working with yaml and properties in Spring Boot.

### Prerequisites

* Java 17
* Maven 3

### Build

```bash
mvn clean package -DskipTests=true
```

### Test

```bash
mvn  test
```

### Run

```bash
mvn  spring-boot:run
```
