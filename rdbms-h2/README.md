# <p align="center">RDBMS H2</p>

## Prerequisites

* Java 17
* Maven 3

## Build

```bash
mvn clean package -DskipTests=true
```

## Test

```bash
mvn  test
```

## Run

```bash
mvn  spring-boot:run
```

see [localhost:8080/h2-console](localhost:8080/h2-console)
