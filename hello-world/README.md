# <p align="center">Hello World</p>

This is the first step of the Spring Boot tutorial. The result is print `Hello World!`.

```java

System.out.println("Hello World!");

```

## Prerequisites

* Java 17
* Maven 3

## Build

```bash
mvn clean package -DskipTests=true
```

## Test

```bash
mvn  test
```

## Run

```bash
mvn  spring-boot:run
```
